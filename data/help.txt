Command Examples:
"move to exit_hatch"
"look at the book"
"get the flare"

Command Key Words:
look - gives you a description of an item
    usage: look {target}
move - how to navigate the world
    usage: move {target}
use - use items that can help you
    usage: use {item} optinal[{ target2 }]
take - get an item from world adds it to inventory
    usage: take {item} optional[{ container }]
drop - put an item back in the environment
    usage: drop {item} optional[{ container }]

Reserved Key Words:
inventory - opens inventory
me - refers to you
help - your here
start - starts game
quit - quits game
