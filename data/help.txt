Command Examples:
"move to exit_hatch"
"look at the book"
"get the flare"

Command Key Words:
look - gives you a description of an item
    usage: look {target}
    synonyms: search, examine, describe
move - how to navigate the world
    usage: move {target}
    synonyms: go, walk, run
use - use items that can help you
    usage: use {item} optinal[{ target }]
    synonyms: activate, interact, touch
take - get an item from world adds it to inventory
    usage: take {item} optional[{ container }]
    synonyms: grab, steal, loot
drop - put an item back in the environment
    usage: drop {item} optional[{ container }]
    synonyms: leave, give, dicard

Reserved Key Words:
inventory - opens inventory
    synonyms: ag, backpack, pocket
me - refers to you
    synonyms: i, player, myself
help - your here
    synonyms: assist, aid, support
start - starts game
    synonyms: begin, init, initiate, execute
quit - quits game
    synonyms: exit, end, terminate
